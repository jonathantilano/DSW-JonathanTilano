Proposito del Sitio:

La finalidad del sitio es fomantar el aprendizaje de las tematicas correspondientes a las "Etiquetas semánticas, CSS Grid" para apropiarnos del aprendizajhe del curso diseño de sitios web del programa de ingenieria de sistemas, atras vez de explorar el curso y el diseño de uma pagina html

Introduccion

Durante el curso de diseño de paginas web, identificamos, reconocoemos y aplicamos los diferenes recursos, para ser aplicados las etiqutas semanticas en HTML y recursos de CSS grip para crear paginas web, utilizando un css-grid-layout, identificando: head, body, header, main, section, article, aside, footer, aplicando clases, background, nav, div, por mencionar algunos ejemplos aplicados para el diseño de esta pagina web tenemos: display: flex; flex-direction: row; justify-content: space-between; background-color: #2C3639; padding: 20px; color: aliceblue; entre otros.

Objetivos

  • Identificar los elementos diferenes etiquetas y recursos para paginas web del curso, en etiquetas semanticas y CSS grip

(Etiquetas semánticas)

Las etiquetas semánticas ayudan a definir la estructura del documento y permiten que las páginas web sean mejor indexadas por los buscadores. Una etiqueta se califica como semántica si tiene que ver el significado, es decir, si nos informa sobre lo que trata su contenido.

(CSS Grid)

El CSS grid se puede utilizar para lograr muchos diseños diferentes. También se destaca por permitir dividir una página en áreas o regiones principales, por definir la relación en términos de tamaño, posición y capas entre partes de un control construido a partir de primitivas HTML
